## 📌 Project Overview

This project analyzes operational performance using **Key Performance Indicators (KPIs)** and **Service Level Agreement (SLA) metrics**.

The dashboard helps monitor:

- Ticket workload trends
- SLA compliance performance
- Team productivity
- Customer satisfaction
- Process efficiency

---

# 🛠 Tools Used

- Power BI
- PostgreSQL
- Excel / CSV
- GitHub
